# Essay Writer

I needed a simple and minimalistic essay writer for getting word count and writing timed essays for my graduate record examinations. So I've build one.  

![sample-image](../assets/images/dp.jpeg)

## Installation

1. clone/download the folder.
2. Install nodejs and npm in you system and use following commands to run it locally.

```bash
cd my-editor
npm install express --save
npm install body-parser --save
```

## Usage
1. type.

```
node index.js
```
2. open chrome and run the server on localhost.
